# 🧮 Calculadora del Siglo

Aplicación desarrollada en Flutter que permite realizar operaciones matemáticas básicas:

- ➕ Suma
- ➖ Resta
- ✖️ Multiplicación
- ➗ División

## ✨ Características

- Interfaz moderna en tonos oscuros
- Diseño intuitivo y elegante
- Historial de operaciones
- Validación de números
- Botón para limpiar datos
- Botón para borrar historial

## Tecnologías utilizadas

- Flutter
- Dart
